## Doxygen configuration
--------------
### General
* Customer: Melexis BU-Actuators
* Description: Default Doxygen configuration files.

### Getting started
Install Doxygen from http://www.stack.nl/~dimitri/doxygen/download.html

Windows users make sure the Doxygen bin folder ends up in your PATH variable.

A Makefile as in `example\Makefile` should be added 1 level above the doxyfile. Project specific definitions inside should be adjusted.
